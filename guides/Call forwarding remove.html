Prefer Using Settings Instead?
📱 iPhone
Go to Settings → Phone → Call Forwarding and toggle it off. If you originally set up forwarding using the code, this toggle may not reflect it — using the code below is more reliable.
📱 Samsung / Android
Open the Phone app → ⋮ menu → Settings → Supplementary services (or Calling accounts) → Call forwarding, then turn off "When unanswered."
Code Didn't Work?
💡 Try This
If your phone still shows forwarding as active after dialling the code, try restarting your phone and dialling it again. If the issue continues, contact your network provider (Vodafone, Three, Eir, etc.) for help.
