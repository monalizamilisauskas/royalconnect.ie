Set Up Call Forwarding

4 simple steps · Takes less than 1 minute

Follow these steps so that calls you don't answer are automatically forwarded to Royal Connect. Your phone will still ring normally — forwarding only kicks in if you don't pick up.
✓ Works on iPhone, Samsung, Pixel & every Android
This code works the same way on every phone. It's quicker and more reliable than digging through Settings menus — just type it into your regular dialer keypad like a phone number.

Prefer Using Settings Instead?
📱 iPhone
Go to Settings → Phone → Call Forwarding. Note: this only offers "forward all calls," so your phone won't ring at all. We recommend using the code above instead.
📱 Samsung / Android
Open the Phone app → ⋮ menu → Settings → Supplementary services (or Calling accounts) → Call forwarding. Some Android phones let you choose "When unanswered" here — if available, use that option and enter your Royal Connect number.
Code Didn't Work?
💡 Try This
In rare cases, some phone plans need forwarding enabled by the carrier first. If nothing happens after dialling the code, contact your network provider (Vodafone, Three, Eir, etc.) and ask them to enable "conditional call forwarding" on your line — then try the code again.
